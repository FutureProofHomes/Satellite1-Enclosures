This is the minimalistic enclosure to place your Sat1.x DevKit into the ceiling<br>
<br>
it is designed to be powered through USB or with our PoE Shoe (est. availabilty end 2026)<br>
This enclosure isn't designed to host an speaker inside.<br>
<br>
You need following extra parts aside the DevKit itself:<br>
- <a href="https://aliexpress.com/item/1005010265420927.html">4x M3x12mm self tapping screws<br>
- <a href="https://aliexpress.com/item/1005010265420927.html">4x M3x4mm self tapping screws<br>
- <a href="https://aliexpress.com/item/1005009490995202.html">4x Downlight Spring Bracket<br>
<br>
For easy printing, use the provided Slicer Project files.<br>
the faceplate mesh is created with a height modifier inside the project.<br>
- use a height modifier from 0.00-2.00mm<br>
- set top and bottom layers to 0<br>
- set infill to 70%<br>
- set infill to tri-hexagon<br>
- set "only one wall on first layer"<br>
<br>
Be aware, your filament flow has to be dialed in.
